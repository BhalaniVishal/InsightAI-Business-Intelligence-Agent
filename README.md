# InsightAI – Business Intelligence & Recommendation Platform

An AI-powered multi-agent system that researches companies, identifies business challenges, recommends AI opportunities, and generates executive-grade intelligence reports.

## Features

- Company Research
- Business Analysis
- Challenge Detection
- AI Opportunity Mapping
- CEO Pitch Generation
- Professional PDF Reports

## Technology Stack

- Streamlit
- Gemini 2.5 Pro
- DuckDuckGo Search
- ReportLab
- Python

## Architecture

Research Agent
→ Business Analysis Agent
→ Challenge Detection Agent
→ AI Opportunity Agent
→ CEO Pitch Agent
→ PDF Report Generator
